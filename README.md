# 🛒 LUXMART — Full Stack E-Commerce Website

A modern and responsive **E-Commerce Platform** built using **Spring Boot**, **React**, and **Supabase**.  
Users can browse products, search items, register/login, manage orders, and enjoy a smooth shopping experience.

---

## ✨ Features

🛍️ Product Listing & Categories  
🔍 Product Search  
👤 Authentication (Register / Login)  
🛒 Cart Management  
📦 Order Tracking  
⚡ REST API Integration  
☁️ Supabase Database  
📱 Responsive Modern UI  

---

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/garvitmalik-123/E-Commerce.git
cd E-Commerce
```

### Backend

```bash
mvn spring-boot:run
```

---

### Frontend

```bash
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create `.env`

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
```
