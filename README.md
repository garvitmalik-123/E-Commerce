# 🛒 E-Commerce Website

A modern full-stack E-Commerce platform built using **Spring Boot**, **React**, and **Supabase** for backend services and database management.

## 🚀 Features

✨ User Authentication  
🛍️ Product Listing & Browsing  
🛒 Shopping Cart Management  
🔍 Search & Filtering  
📱 Responsive UI  
⚡ REST API Integration  
☁️ Supabase Database Connectivity  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Spring Boot
- REST API
- Maven

### Database & Services
- Supabase

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/garvitmalik-123/E-Commerce.git
cd E-Commerce
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

Create `.env` file:

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
```
